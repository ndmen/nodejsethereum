# nodejsethereum

This application receives balance from an Ethereum address

## Installation

Clone this code and install modules:

```sh
npm install web3 typescript tslint ts-node nodemon fs express dotenv cors @types/node @types/express
```

## Usage

Run application:

```sh
npm run dev
```
